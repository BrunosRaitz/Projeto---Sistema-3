# Projeto---Sistema-3

###Alarme de incêndio###

descrição do grupo: O sistema anti-incêndio funciona de forma automática: ele inicia verificando os sensores, entra em um loop contínuo e coleta dados de temperatura e fumaça. Em seguida, compara esses valores com limites seguros. Se detectar risco, aciona alarmes e envia notificações; em casos graves, ativa o sistema de combate, como sprinklers. Todas as ações são registradas, e o sistema continua monitorando constantemente.


Lista componentes:
Arduino UNO;
Sensor de fumaça;
